# Main Objective

Design a modern, clean, and visually engaging web dashboard for a service called “Legendary Profiles.” The dashboard should feel playful but professional, with subtle fantasy‑themed accents (think Magic‑the‑Gathering‑inspired UI hints, not full medieval cosplay). Prioritize usability and clarity.

# Core Dashboard Sections (must be immediately visible or one click away):

    - My Cards

        Display all user‑created cards in a grid or gallery view.

        Each card should show a thumbnail of the card design.

        Include quick‑action buttons on each card: “Edit,” “Order,” “Share,” and “View QR.”

        A small card or banner showing whether the user has the basic or premium QR code features.

        If basic, include an upgrade CTA.

        Provide a prominent “Create New Card” button at the top.

    - Create a Card

        A clear call‑to‑action button or panel that stands out visually.

        Should feel like the primary action of the dashboard.

        Use iconography or color to make it feel like a “Start Your Adventure” moment.

    - Orders

        A section showing current orders, their status (e.g., “Processing,” “Shipped,” “Delivered”), and estimated delivery dates.

        A history of previous orders with thumbnails of the cards ordered.

        Include a “Reorder” button for past purchases.

    - Quick Actions Panel

        A small sidebar or top‑bar area with shortcuts:

            “Create New Card”

            “Order Cards”

            “Manage Subscription”

            “View Link Hub” (if subscribed)

Design Style Guidelines:

    Clean, modern layout with soft shadows and rounded corners.

    Color palette inspired by fantasy card games: deep blues, gold accents, subtle gradients.  You can change the CSS of the entire website to match if you would like.

    Typography should be bold and readable, with a touch of personality.

    Use card‑like UI elements to reinforce the theme.

    Include small whimsical touches (e.g., icons that look like mana symbols, subtle sparkles, or parchment textures) without overwhelming the modern aesthetic.

Overall Tone:

    Friendly, adventurous, slightly humorous.

    Should feel like a cross between a professional dashboard and a character‑creation screen.
